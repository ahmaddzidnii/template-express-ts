## Installation

1. clone this repository

   ```shell
    git clone https://github.com/ahmaddzidnii/template-express-ts.git
   ```

2. Install the required dependencies

   ```shell
   npm install
   ```

3. Start your server

   ```shell
   npm run dev
   ```

4. Done, check the localhost port 2000 in the browser

   ```json
   {
     "message": "Hello World!",
     "NODE_ENV": "development"
   }
   ```
